# アイコン画像について

この拡張機能のアイコンは、以下の仕様で作成してください。

---

## 🎨 デザイン要件

### 禁止事項
- **OpenAI / ChatGPTのロゴや配色を使用しない**
- OpenAIの緑色（#10A37F）は避ける
- ChatGPTのロゴマーク（円形のアイコン）に似たデザインは避ける

### 推奨デザイン
- **中立的な幾何学デザイン**
- 削除機能を連想させるシンプルなアイコン
- 色: ブルー系、グレー系、レッド系（警告色として）

---

## 📐 サイズと形式

| ファイル名 | サイズ | 用途 |
|----------|------|------|
| icon16.png | 16x16px | ブラウザのツールバー（小） |
| icon48.png | 48x48px | 拡張機能管理ページ |
| icon128.png | 128x128px | Chrome Web Store |

- **形式**: PNG（透過背景推奨）
- **カラースペース**: sRGB
- **DPI**: 72dpi以上

---

## 💡 デザイン案

### 案1: チェックボックスとゴミ箱
- 複数のチェックボックス（選択機能を表現）
- ゴミ箱アイコン（削除機能を表現）
- 色: ダークブルー (#1e40af) とライトグレー (#d1d5db)

### 案2: リストと消しゴム
- リスト形式のライン（複数行）
- 消しゴムアイコン（削除を表現）
- 色: ダークグレー (#374151) とレッド (#ef4444)

### 案3: 抽象的な削除マーク
- 複数の要素を×で消す表現
- ミニマルな幾何学デザイン
- 色: ネイビー (#1e3a8a) とオレンジ (#f97316)

---

## 🛠️ 作成方法

### オプション1: Figma / Adobe Illustrator
1. 128x128pxのキャンバスを作成
2. 上記のデザイン案を参考に作成
3. PNG形式でエクスポート
4. 48x48px、16x16pxにリサイズ

### オプション2: Canva（無料）
1. カスタムサイズ（128x128px）でデザインを作成
2. シンプルな図形（四角、円、線）を組み合わせる
3. PNG形式でダウンロード
4. オンラインツール（例: TinyPNG）でリサイズ

### オプション3: DALL-E / Midjourney（AI生成）
```
プロンプト例:
"Simple geometric icon for a browser extension, representing bulk deletion. 
Minimalist design, blue and gray colors, 128x128px, flat design, 
no text, no ChatGPT logo, neutral appearance"
```

生成後、背景を透過してPNG形式で保存。

---

## 📦 配置

作成したアイコンファイルを以下に配置してください：

```
icons/
├── icon16.png   (16x16px)
├── icon48.png   (48x48px)
└── icon128.png  (128x128px)
```

---

## ✅ 確認事項

アイコン作成後、以下を確認してください：

- [ ] OpenAI / ChatGPTのロゴ・配色に似ていない
- [ ] 透過背景（または単色背景）
- [ ] 解像度が適切（16px, 48px, 128px）
- [ ] PNG形式
- [ ] ファイルサイズが適切（各ファイル100KB以下推奨）
- [ ] 視認性が高い（小さいサイズでも認識できる）

---

## 🎨 仮アイコンの使用（開発中）

開発中は、シンプルな単色の正方形や円などを使用しても構いません。

Chrome Web Store公開時には、必ず適切なアイコンに差し替えてください。
